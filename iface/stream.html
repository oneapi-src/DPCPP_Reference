Streams¶

Kernels may not use std streams for input/output. sycl::stream provides similar functionality.

Example

Output to stdout in a kernel.

#include <CL/sycl.hpp>

int main() {
  sycl::queue q;
  q.submit([&](sycl::handler &h) {
	     // setup sycl stream class to print standard output from
	     // device code
	     auto out = sycl::stream(1024, 768, h);
	     auto task =
	       [=]() {
		 out << "In a task\n";
	       };
	     h.single_task(task);
	   });
}
